# 🌑 Dark Survival Modpack

Um modpack do CurseForge para Minecraft projetado para transformar o jogo em uma experiência mais sombria e hostil.  
O objetivo é manter o jogador sob constante tensão, eliminando qualquer sensação real de segurança — seja em terra, na água, no céu, durante o dia ou na calada da noite.

---

## 📋 Requisitos

Para executar este modpack corretamente, certifique-se de que seu ambiente atende aos seguintes requisitos:

- **Java:** Versão 17 (recomendado: 17.0.x)  
- **RAM:** 2–3 GB alocados para Minecraft  
- **Launcher:** Um launcher compatível com CurseForge  
  - Recomendado: Prism Launcher  

---

## 🚀 Instalação

Siga os passos abaixo para instalar o modpack:

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```

2. Comprima o conteúdo do repositório em um arquivo `.zip`  
   > ⚠️ Certifique-se de compactar os **arquivos dentro do repositório**, e não a pasta raiz.

3. Abra seu launcher (recomendado: Prism Launcher)

4. Importe o modpack:
   - Procure por uma opção como **"Import from ZIP"** ou **"Add Instance from File"**
   - Selecione o arquivo `.zip` que você criou

---

### ⚠️ Nota Importante (Download pelo GitHub)

Se você baixar o projeto usando o botão **"Download ZIP"** do GitHub:

- O GitHub adiciona automaticamente uma pasta extra envolvendo o projeto  
- Esse formato **não é compatível diretamente** com a importação no launcher

## 🤝 Contribuindo

Contribuições são bem-vindas! Para garantir consistência e compatibilidade, siga o fluxo abaixo.

### 📦 Fluxo de trabalho

1. Importe o modpack em um launcher compatível com CurseForge  
   (recomendado: Prism Launcher)

2. Faça suas alterações:
   - Adicione ou atualize mods (**apenas mods disponíveis no CurseForge**)  
   - Modifique arquivos de configuração conforme necessário  

3. Exporte o modpack:
   - Use a funcionalidade de **Exportar** do launcher  
   - Isso irá gerar um arquivo `.zip` no formato de modpack do CurseForge  

4. Prepare sua contribuição:
   - Extraia o `.zip` gerado  
   - Copie o conteúdo para o repositório  
   - ⚠️ **Não envie o arquivo `.zip` diretamente**

5. Abra um Pull Request:
   - Descreva claramente o que foi adicionado/modificado e o motivo  

---

### ⚠️ Diretrizes

- Inclua apenas mods disponíveis no CurseForge  
- Mantenha a experiência alinhada com o tema central do modpack
